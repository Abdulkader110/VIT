# HTML Website Project

This project is a simple website built using HTML, CSS, and JavaScript, inspired by the design of the GDSC VIT Pune website.

## Project Structure

```
html-website-project
├── assets
│   ├── css
│   │   └── styles.css
│   ├── js
│   │   └── scripts.js
│   └── fonts
├── index.html
└── README.md
```

### Files Description

- **index.html**: The main HTML file that serves as the entry point for the website. It contains the structure of the webpage, linking to the CSS and JavaScript files, and includes the content such as headers, sections, and footers.

- **assets/css/styles.css**: This file contains the CSS styles for the website, defining the layout, colors, fonts, and other visual aspects to match the design of the reference website.

- **assets/js/scripts.js**: This file includes JavaScript code for any interactive features of the website, such as navigation, animations, or form handling.

- **assets/fonts**: This directory holds any custom font files used in the website to ensure consistent typography.

## Setup Instructions

1. Clone the repository or download the project files.
2. Open the `index.html` file in a web browser to view the website.
3. Modify the CSS in `assets/css/styles.css` to customize the appearance.
4. Add any interactive features in `assets/js/scripts.js`.

## Design Choices

The design of this website is inspired by the GDSC VIT Pune website, focusing on a clean layout, responsive design, and user-friendly navigation. The color scheme and typography have been chosen to enhance readability and visual appeal.
